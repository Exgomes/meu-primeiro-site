Falando sobre Tags

(voltar)

Temos varias Tags, porém vou citar algumas e suas funcionalidades. Temos a Tag "i" que é utilizada para deixar o texto em itálico. A tag "strong" que deixa o texto em negrito. Você também pode mesclar as duas e usar o "i" e o "strong" para deixar seu texto em itálico e negrito ao mesmo tempo. O "input" é uma tag que ela não tem fechamento como as outras, no final ela fecha sozinha com /> . Ela é uma tag utilizada para criar um campo para digitar textos. Temos o "input type= "text""" para criar uma campo de digitação de textos e o "input type= "number"" para digitação de números. "img" é uma tag para imagem, ela também não precisa ser fechada, e uma diferença que ela tem, é que ela possui um atributo. Esse atributo seria uma configuração para que apareça de uma maneira diferente no navegador. Temos também o "width" que é para a largura da imagem. Os "H1,H2,H3,H4..." para definir o tamanho dos titulos. Se você quiser utilizar um simbolo de raiz quadrada por exemplo existe a tag "sup" que deixaria seu texto assim: 3ª.

Atributo das Tags

(voltar)

Atributo das tags são as propriedades que ela pode ter, por exemplo na tag "strong" ela pode ter um id que seria um identificador, que se for usar o javascript ele consegui localizar esse identificador e trocar o conteúdo dele. Temos também um atributo chamado "Style" esse já é um atributo que é utilizado no CSS para formatar a página.

Textos

(voltar)

Para os texto nós temos vária tipografias, elas são representadas por H1, H2, H3, H4, H5, H6, Cada uma delas define um tamanho de fonte. Agora quando você quer fazer uma citação no texto, trazer uma frase de algum autor pode usar o "blockquote" . Temos o "u" para sublinhar o texto e o "mark" que faz amarcação do texto. Temos o "p" que é para o parágrafo.

Lista ordenadas e desordenadas

(voltar)

No HTML temos dois tipos de listas, as ordenadas e as desordenadas. A tag para uma lista ordenada é "ol" ela vai enumerar a lista. já a tag para lista desordenada seria o "ul" ele vai pontuar a lista.
